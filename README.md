# Veebilehe projekt

## Mida see leht tutvustab?
Võrkpallitrenni teemaline veebileht, kus on teavet võrkpallitreeningu võimaluste kohta Tartus,
erinevaid soojendusharjutuste soovitusi, mängureegleid ning näpunäiteid mängimiseks ja
pildigalerii. Esilehel on üldine teave võrkpalli kohta ning navigatsiooniribal on lingid teistele
lisalehtedele, kust leiab vastavat infot.
